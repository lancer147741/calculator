Этот проект — простой веб-калькулятор, разработанный как pet project для практики HTML, CSS и JavaScript.
Он позволяет пользователю выполнять базовые арифметические операции:

Сложение, вычитание, умножение, деление
Ввод многозначных чисел и десятичных
Очистку дисплея (AC) и удаление последнего символа (⌫)
Просмотр истории последних вычислений
Цель проекта — отработать навыки фронтенд-разработки, работу с DOM, модуляризацию кода и создание адаптивного интерфейса, который выглядит как настоящий калькулятор.

js/calculator.js
-
Основная логика вычислений
Содержит функции: calculate(), append(), clearAll(), deleteLast()

js/ui.js
-
Управляет дисплеем и взаимодействием кнопок с пользователем
Функции для обновления экрана и передачи ввода в calculator.js

js/history.js
-
Ведение и отображение истории вычислений на странице
Использует список <ul> для добавления новых записей

js/app.js
-
Инициализация приложения
Можно использовать для подключения дополнительных функций

Функциональность
-
Пользователь может вводить числа и операции с клавиатуры интерфейса
Нажатие = выполняет расчет
Нажатие AC очищает дисплей
Нажатие ⌫ удаляет последний символ
История вычислений сохраняется и отображается в нижней части калькулятора

Технологии
-
HTML5
CSS3 (Grid Layout, стилизация кнопок, адаптивность)
Vanilla JavaScript (модульная структура, DOM-манипуляции)
